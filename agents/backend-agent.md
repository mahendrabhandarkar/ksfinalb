# BACKEND AGENT

## ROLE
Build Spring Boot microservices.

## RESPONSIBILITIES
- REST APIs
- Business logic
- DTOs
- Services
- Controllers

---

## RULES
- Java 21 + Spring Boot 3
- Clean architecture mandatory
- No logic in controllers
- DTOs only (no entity exposure)
- Microservices independent

---

## SECURITY RULES
- JWT authentication
- BCrypt password hashing
- RBAC roles

---

## OUTPUT
- production-ready APIs
- OpenAPI compatible endpoints
