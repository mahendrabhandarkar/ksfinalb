# UI/UX SPEC - MATRIMONIAL PLATFORM

## DESIGN GOAL
Premium matrimonial experience like modern SaaS apps (clean, trustworthy, fast).

---

## UI STYLE

- Minimal design
- High trust visual language
- Soft shadows
- Rounded cards
- Consistent spacing system (4px grid)

---

## COLOR SYSTEM

Primary: Indigo / Blue
Secondary: Gray scale
Success: Green
Warning: Orange
Danger: Red

Dark mode mandatory.

---

## TYPOGRAPHY

- Font: Inter / system UI
- Clear hierarchy:
  H1 → H2 → H3 → body → caption

---

## COMPONENTS

Must include:

- Profile card
- Search filters panel
- Match recommendation grid
- Chat UI
- Subscription pricing cards
- Verification badge UI

---

## RESPONSIVE RULES

- Mobile first
- No horizontal scroll allowed
- Touch targets >= 44px
- Sidebar collapses on mobile

---

## UX RULES

- Fast onboarding (max 3 steps)
- Clear CTA buttons
- Minimal friction registration
- Instant feedback on actions
